Setup at a glance

  • MCP (preferred): set transport=http + the server base_url; the token goes in secret (sent as a Bearer header). Unlimited platforms.
  • Built-in (backup): leave transport blank for the bundled GitHub/Jira/Plane clients.
  • Roles: a connector can be a source, a sink, or both — a default sink must also be a sink, and only github/jira/plane can be sources.
  • Config examples — github {"repo":"owner/name"}, jira {"email":"…","project_key":"ENG"}, plane {"workspace_slug":"…","project_id":"…"}.
