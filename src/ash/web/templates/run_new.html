To pull an issue from GitHub, Jira, or Plane you need a connector. Pick one path:

Path A — add a connector in Admin (recommended):

  1. Go to /admin → Connectors → + Add.
  2. Set Kind = github (or jira / plane).
  3. Set Secret = your GitHub Personal Access Token (needs repo scope for private repos; public repos need read:public_repo or no auth).
  4. In Config, add {"repo": "owner/name"} — e.g. {"repo": "makeplane/plane"}.
  5. Check Is source and Enabled, then save.
  6. Come back here — the connector will appear in the "Issue source" dropdown.
Path B — use the project YAML (no DB connector needed): Leave the "Issue source" blank. ASH will use issues.source_repo from projects/<name>.yaml + GITHUB_TOKEN from .env. Only works for GitHub; the project YAML must have issues: source_repo: owner/name.
